Web TradingApp

This sample illustrates the ability to add a responsive web page to a TradeStation TradingApp. It is also possible to communicate via hash tags and parameters in a URL. Click on to place a trade in the platform.

If you are using TradeStation 9.1 Update 24, this browser window will be based on Microsoft Internet Explorer 7. In Update 25, we will support up to IE 10 (based on the browser installed on the user's system). If you would like to emulate the functionality of Update 25 in Update 24 update your registry with the following values:

[HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Internet Explorer\MAIN\FeatureControl\FEATURE_BROWSER_EMULATION]
  
"TSElAppHost.exe"=dword:00002710
